# React + Vite + Eslint + Husky

> Start dev server

```shell
npm run dev
```

> Build project

```shell
npm run build
```

> Lint

```shell
npm run lint
```

> Lint fix

```shell
npm run lint:fix
```

> Prepare to push on VCS

```shell
npm run prepare
```

> Write prettier

```shell
npm run pretty
```
